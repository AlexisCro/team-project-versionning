Réalisme

Le réalisme est le maître mot de notre jeu

UltimateRace Simulation est un jeu de simulation de course automobile. Il est développé par notre équipe de passionnés de course automobile et de jeux vidéo. Nous avons voulu créer un jeu de simulation de course automobile qui soit le plus réaliste possible. Nous avons donc modélisé les voitures et les circuits avec un grand réalisme. Nous avons également voulu que le jeu soit accessible à tous.

  • Pour qui ? Tous les passionnés de course automobile et de jeux vidéo
  • Catégorie : SimRacing
  • Plateformes disponibles : PC (Steam), PS4/5 et XBOX
  • Langues disponibles : Français, Anglais, Allemand, Espagnol, Italien, Portugais, Russe, Chinois, Japonais
  • Configuration PC conseillée :
    • OS : Windows 10 64 bits
    • Processeur : Intel Core i5-9600K ou AMD Ryzen 5 2600X
    • RAM : 16 Go
    • Carte graphique : Nvidia GeForce GTX 1660 ou AMD Radeon RX 590
    • Espace disque : 50 Go
    • Volant et pédalier conseillé
